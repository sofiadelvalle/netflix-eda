# Netflix Titles — Exploratory Data Analysis

## Descripción

Este proyecto realiza un análisis exploratorio del catálogo de Netflix con el objetivo de identificar patrones en la distribución del contenido, su evolución temporal, el origen geográfico de los títulos, los géneros predominantes, la clasificación por edad (rating) y la duración de las películas.

A través de técnicas de limpieza y transformación de datos con Python (pandas), se extraen insights relevantes que permiten comprender mejor la estrategia de contenido de la plataforma.

## Objetivo

El análisis se centra en:

- Evaluar la distribución del catálogo entre películas y series.
- Analizar la evolución anual de incorporación de títulos.
- Identificar los países con mayor representación en la plataforma.
- Estudiar los géneros predominantes.
- Examinar la orientación del contenido según clasificación por edad.
- Analizar la duración de las películas y su relación con el rating.
